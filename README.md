# Shakespear
# Shakespeare Erasmus Project

## Overview
A multilingual interactive Shakespeare experience for the Erasmus project, featuring chatbot interactions, quiz-based character discovery, events, memorial garden, and global cultural exploration.

## Features
- Shakespeare character quiz.
- Shakespeare chat.
- Event discovery.
- Digital memorial garden.
- 3D globe exploration.
- Feedback and session tracking.

## Setup
1. Clone the repository.
2. Install dependencies.
3. Add your .env file.
4. Set up the database.

## Run the Project
Backend:
```bash
npm install
npm start
```

Frontend:
```bash
npm run dev
```
